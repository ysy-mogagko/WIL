### 개인 프로젝트
- pprof를 이용하여 CPU, Memory 프로파일링
```
package main

import (
  "net/http"
  _ "net/http/pprof"
)
func main() {
  go func() {
    http.ListenAndServe("localhost:6060", nil)
  }()
  // 애플리케이션 로직
  select {}
}
```
- 설정 이후 curl 명령어로 프로파일링 데이터 수집
```
curl http://localhost:6060/debug/pprof/profile?seconds=30 --output heap.prof 
```
